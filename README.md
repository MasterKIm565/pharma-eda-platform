## 📌 Project Overview

A comprehensive web-based statistical analysis platform designed for pharmaceutical quality control professionals. 
This tool provides essential statistical methods required for CMC (Chemistry, Manufacturing, and Controls) comparability studies, 
process validation, and regulatory submissions.

**Key Highlight:** AI-powered result interpretation using Google Gemini API for real-time statistical guidance.

---

## ✨ Features

### Core Statistical Analyses
- **TOST (Two One-Sided Test)** - Equivalence testing for tech transfer and comparability studies
- **ANOVA** - Analysis of variance for multi-group comparisons
- **SPC (Statistical Process Control)** - Real-time process monitoring with control charts
- **Process Capability** - Cp, Cpk analysis for validation studies

### Advanced Features
- **Data Explorer** - 6 visualization types (Histogram, Box Plot, Scatter, Bar, Line, Violin)
- **Multi-Variable Analysis** - Dynamic variable addition with correlation and trend analysis
- **AI Assistant** - Google Gemini integration for automated result interpretation
- **Interactive Visualizations** - Powered by Plotly.js for publication-quality charts
